## Deploy Your Own


## Developing Locally

You can clone & create this repo with the following command

```bash
yarn install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:5000](http://localhost:5000) with your browser to see the result.
